# Blueprint

## Overview

This project is a Next.js application for a mentorship platform. It allows students to find and schedule mentorship sessions with professors. The application is built with the App Router and uses Server Components for data fetching and Server Actions for mutations.

## Features

- User authentication (student and professor roles)
- View available mentors
- Schedule mentorship sessions
- View upcoming and past mentorships
- Rate and review mentors

## Design

- **Styling:** Tailwind CSS
- **Components:** Shadcn/ui
- **Icons:** Lucide React
